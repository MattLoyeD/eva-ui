# Sync Ratio Chart

Pure SVG dual sinusoidal waveform chart. Zero external charting dependencies. Uses a precise `generateSineWavePath` function with 100 sample points per wave.

```tsx
import { SyncRatioChart } from "@/components";
```

## Basic Usage

<ComponentPreview
  title="DEFAULT"
  code={`<div style={{ height: "250px" }}>
  <SyncRatioChart showGrid animated />
</div>`}
>
  <div style={{ height: "250px" }}>
    <SyncRatioChart showGrid animated />
  </div>
</ComponentPreview>

## Props

<PropsTable
  props={[

<Callout type="info" title="WAVEFORM MATH">
  Each wave is computed as: `y = centerY + amplitude * Math.sin(x * frequency + phase)` across 100 evenly-spaced sample points. Wave A renders in cyan, Wave B in magenta.
</Callout>
