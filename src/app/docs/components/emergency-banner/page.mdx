# Emergency Banner

Full-screen or inline alert banner with animated hazard stripes, scanline overlay, and flickering text. Used for PATTERN BLUE detection, system failures, and general alerts.

```tsx
import { EmergencyBanner } from "@/components";
```

## Basic Usage

<ComponentPreview
  title="DEFAULT"
  code={`<EmergencyBanner
  text="EMERGENCY"
  subtext="PATTERN BLUE DETECTED — ALL PERSONNEL TO BATTLE STATIONS"
  visible
/>`}
>
  <EmergencyBanner
    text="EMERGENCY"
    subtext="PATTERN BLUE DETECTED — ALL PERSONNEL TO BATTLE STATIONS"
    visible
  />
</ComponentPreview>

## Accessibility

<Callout type="info" title="ARIA ATTRIBUTES">
  EmergencyBanner uses `role="alert"` and `aria-live="assertive"` to announce the alert to screen readers immediately when it becomes visible.
</Callout>

The flickering text animation respects `prefers-reduced-motion` — when reduced motion is preferred, the text displays without animation.
