# System Dialog

Modal dialog with hex grid background overlay. Severity levels add red borders, blinking warning icons, and hazard stripes. Uses `AnimatePresence` for smooth enter/exit transitions.

```tsx
import { SystemDialog } from "@/components";
```

## Severity Levels

<Callout type="warning" title="INTERACTIVE COMPONENT">
  SystemDialog is a controlled modal that requires `open` state and callback handlers. The previews below show the dialog content styling without the full overlay behavior.
</Callout>

### Normal

```tsx
<SystemDialog
  open={isOpen}
  title="SYSTEM NOTIFICATION"
  severity="normal"
  onAccept={() => setOpen(false)}
  onDecline={() => setOpen(false)}
>
  <p>Routine maintenance scheduled for 0300 hours.</p>
</SystemDialog>
```

### Warning

```tsx
<SystemDialog
  open={isOpen}
  title="OVERRIDE REQUEST"
  severity="warning"
  acceptText="AUTHORIZE"
  declineText="ABORT"
  onAccept={handleOverride}
  onDecline={handleCancel}
>
  <p>MAGI authorization required for Unit-01 activation.</p>
</SystemDialog>
```

### Critical

```tsx
<SystemDialog
  open={isOpen}
  title="SELF-DESTRUCT CONFIRMATION"
  severity="critical"
  acceptText="CONFIRM"
  declineText="ABORT"
  onAccept={handleConfirm}
  onClose={handleClose}
  showHazardStripes
>
  <p>This action is irreversible. All personnel must evacuate.</p>
</SystemDialog>
```
