# Eva Tooltip

Tactical overlay tooltip with L-bracket corner accents and configurable placement. Wraps any trigger element and displays content on hover/focus with an optional delay.

```tsx
import { EvaTooltip, Button } from "@mattloyed/eva-ui";
```

## Basic Usage

<ComponentPreview
  title="BASIC"
  code={`import { EvaTooltip, Button } from "@mattloyed/eva-ui";

<EvaTooltip content="LAUNCH SEQUENCE READY">
  <Button>Hover me</Button>
</EvaTooltip>`}
>
  <EvaTooltip content="LAUNCH SEQUENCE READY">
    <Button>Hover me</Button>
  </EvaTooltip>
</ComponentPreview>

  The tooltip uses `aria-describedby` and `role="tooltip"` for screen reader support. It responds to both hover and focus events.
