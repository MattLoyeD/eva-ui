# Eva Theme Provider

Wrapper component that injects CSS custom properties onto a `<div>`, allowing descendant components to inherit overridden theme values. Keys without a `--` prefix are automatically prefixed with `--color-`.

```tsx
import { EvaThemeProvider } from "@mattloyed/eva-ui";
```

## Basic Theme Override

<ComponentPreview
  title="CUSTOM COLORS"
  code={`import { EvaThemeProvider, EvaCard, Button } from "@mattloyed/eva-ui";

<EvaThemeProvider
  theme={{
    "eva-green": "#ff00ff",
    "eva-orange": "#00ffcc",
  }}
>
  <EvaCard title="THEMED PANEL">
    <div className="text-sm font-mono text-eva-white/80 space-y-2">
      <div>Custom properties cascade to all children.</div>
      <Button className="mt-2">THEMED BUTTON</Button>
    </div>
  </EvaCard>
</EvaThemeProvider>`}
>
  <EvaThemeProvider
    theme={{
      "eva-green": "#ff00ff",
      "eva-orange": "#00ffcc",
    }}
  >
    <EvaCard title="THEMED PANEL">
      <div className="text-sm font-mono text-eva-white/80 space-y-2">
        <div>Custom properties cascade to all children.</div>
        <Button className="mt-2">THEMED BUTTON</Button>
      </div>
    </EvaCard>
  </EvaThemeProvider>
</ComponentPreview>

## Raw CSS Variables

Use the `--` prefix to set arbitrary CSS custom properties directly.

<ComponentPreview
  title="RAW VARIABLES"
  code={`<EvaThemeProvider
  theme={{
    "--font-eva-display": "monospace",
    "--custom-glow": "rgba(255, 0, 128, 0.3)",
  }}
>
  <EvaCard title="RAW OVERRIDES">
    <div className="text-xs font-mono text-eva-white/60">
      Keys starting with -- are passed through as-is.
    </div>
  </EvaCard>
</EvaThemeProvider>`}
>
  <EvaThemeProvider
    theme={{
      "--font-eva-display": "monospace",
      "--custom-glow": "rgba(255, 0, 128, 0.3)",
    }}
  >
    <EvaCard title="RAW OVERRIDES">
      <div className="text-xs font-mono text-eva-white/60">
        Keys starting with -- are passed through as-is.
      </div>
    </EvaCard>
  </EvaThemeProvider>
</ComponentPreview>

<Callout type="info" title="KEY PREFIXING">
  Keys without a `--` prefix are automatically converted to `--color-{key}`. For example, `"eva-green": "#ff00ff"` becomes `--color-eva-green: #ff00ff`. Keys that already start with `--` are passed through unchanged.
</Callout>
