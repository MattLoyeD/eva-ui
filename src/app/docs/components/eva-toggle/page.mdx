# Eva Toggle

Sliding ON/OFF switch with glowing indicator and monospaced status text. Uses `role="switch"` for accessibility. Requires controlled state via `checked` and `onChange`.

```tsx
import { EvaToggle } from "@mattloyed/eva-ui";
```
